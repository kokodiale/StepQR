Generator Kodów QR - Aplikacja Webowa
====================================

Aplikacja webowa do generowania kodów QR z linków i ścieżek Dropbox.

Wymagania
---------
- Python 3.8 lub nowszy
- Flask
- qrcode
- Pillow
- python-dotenv

Instalacja
----------
1. Sklonuj repozytorium
2. Zainstaluj wymagane pakiety:
   pip install -r requirements.txt

Uruchomienie
-----------
1. Uruchom aplikację:
   python app.py
2. Otwórz przeglądarkę i przejdź pod adres:
   http://localhost:5000

Funkcjonalności
--------------
- Generowanie kodów QR z dowolnych linków
- Generowanie kodów QR z linków Dropbox
- Automatyczna modyfikacja linków Dropbox dla bezpośredniego pobierania
- Możliwość pobierania wygenerowanych kodów QR
- Responsywny interfejs użytkownika

Użycie
------
1. Wprowadź link w pole "Link do wygenerowania kodu QR"
2. Opcjonalnie wprowadź ścieżkę Dropbox
3. Kliknij "Generuj kod QR"
4. Pobierz wygenerowany kod QR używając przycisku "Pobierz kod QR"
